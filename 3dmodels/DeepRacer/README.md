Este es el directorio que contiene los archivos del DeepRacer para Blender y CARLA. 

---

### DeepRacer en Blender

El archivo [DeepRacer.blend](/DeepRacer/blender/DeepRacer.blend) contiene el modelo completo del DeepRacer, ya configurado con todos sus enlaces y articulaciones correctamente definidos, utilizando exactamente los mismos nombres que CARLA requerirá posteriormente para identificar correctamente cada parte del vehículo.

![DeepRacer en Blender](/DeepRacer/media/blendernormal.png)

Puedes encontrar un tutorial sobre cómo crear toda la configuración desde cero en este [enlace](https://urjc-deepracer.github.io/docs/createdeepracerinblender/), o bien ver el siguiente vídeo en YouTube:
