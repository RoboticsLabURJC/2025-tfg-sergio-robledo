# TFG Sergio Robledo Sarrión
Bienvenido al repositorio del TFG titulado: **Diseño, implementación y evalucación de un sistema de conducción autónoma para entornos de competición basado en aprendizaje por imitación en CARLA**

Este repositorio cuenta con dos bloques principales:

- En primer lugar el modelado y simulación del vehículo AWS Deepracer, así como de circuitos. Todo ello en el directorio de [3dmodels](/3dmodels). 
En el video se muestra el comportamiento del vehículo una vez importado a CARLA de manera correcta:

[manualcontrol.webm](https://github.com/user-attachments/assets/d4cc3f78-3c9f-40a5-a193-f2b679dd2997)

- En segundo lugar, las arquitecturas de aprendizaje por imitación, así como las pruebas realizadas en el simulador en inferencia, que se encuentran en el directorio de [scripts](/scripts). 
En el video se muestra el AWS  conduciendo de forma autónoma en inferencia en uno de los circuitos:
